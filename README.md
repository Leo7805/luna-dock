# Luna Dock

A modern Chrome extension that enhances AI chat platforms with a smart sidebar and navigation dock.

## Features

- Modern sidebar UI
- AI conversation navigation
- Floating dock button
- ChatGPT support (WIP)
- Copilot support (Planned)
- Gemini support (Planned)

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- Chrome Extension (Manifest V3)
